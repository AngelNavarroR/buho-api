# buho-api
Permite realizar consultas usando JPA pasando el nombre de la entidad, devuelve un json los campos de la entidad

# Api 
## Busquedas
Todas las api de busquedas usan el modelo BusquedaModel y reciben un formato json
### /filters/findBy
### /filters/findBy/page
### /filters/findBy/count
### /filters/findBy/max
### /filters/findBy/exists

## Guardado
Usan el modelo GuardarModel
### /filters/save/entiti
### /filters/save/entiti/map

## Eliminar
Usan el modelo GuardarModel
### /filters/eliminar/entiti
